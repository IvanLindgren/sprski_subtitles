# Читавук-речник

Читавук-речник — веб-сервис для создания сербских субтитров, синхронного просмотра видео и ведения личного словаря. Пользователь загружает видео, получает текст с таймкодами, сохраняет незнакомые слова с переводами и заметками, а затем скачивает VTT, SRT или готовый MP4 со встроенными субтитрами.

## Локальный запуск

Для запуска нужен Node.js. Выполните `npm install`, затем скопируйте `.env.example` в `.env` и при желании добавьте ключ из [Groq Console](https://console.groq.com/keys). После команды `npm run dev` сайт откроется по адресу [http://127.0.0.1:5173](http://127.0.0.1:5173). Ключ также можно ввести непосредственно в интерфейсе, где он хранится только до закрытия вкладки.

## Возможности

Сервис принимает MP4, MOV, WEBM, MKV и другие форматы, извлекает речевую дорожку и отправляет её в `whisper-large-v3` с языком `sr`. Временная шкала аудио предварительно обнуляется, а если основное распознавание пропустило начало, первые секунды проверяются отдельным запросом. Проекты сохраняются в LocalStorage, исходные видео размещаются в IndexedDB браузера, а системная установка FFmpeg не требуется благодаря `ffmpeg-static`.

## Бесплатный тариф

На июль 2026 года Groq публикует для Whisper лимит в 2 000 запросов в сутки, 7 200 секунд аудио в час и 28 800 секунд в сутки. Прямое вложение ограничено 25 МБ, поэтому сервер предварительно преобразует дорожку в моно-аудио Opus с частотой 16 кГц. Актуальные значения находятся в документации [Groq rate limits](https://console.groq.com/docs/rate-limits) и [Speech to Text](https://console.groq.com/docs/speech-to-text).

## Деплой на Render

Репозиторий содержит `render.yaml`, поэтому его можно развернуть как Blueprint через [Render Dashboard](https://dashboard.render.com/blueprints). Render установит зависимости, соберёт React-интерфейс, запустит Express-сервер и будет проверять `/api/health`. В настройках Blueprint нужно указать `GROQ_API_KEY`, если предполагается общий серверный ключ. Без серверного ключа каждый пользователь сможет вводить собственный ключ в интерфейсе. Бесплатный web service Render засыпает после пятнадцати минут без запросов и запускается снова при следующем обращении.
